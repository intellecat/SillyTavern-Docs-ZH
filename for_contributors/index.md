# 开发与自动化

SillyTavern 的默认安装包仅提供一个基础平台，可通过多种方式进行扩展。

:::callout
**[STscript](st-script)**

STscript 是一种功能强大的脚本语言，基于批量聊天命令构建，无需任何编程基础即可上手使用。
:::

:::callout
**[函数调用（Function Calling）](Function-Calling)**

通过让 LLM 使用外部数据源或触发扩展的特定功能，为其添加更动态的能力。
:::

:::callout
**[UI 扩展（UI Extensions）](Writing-Extensions)**

UI 扩展在浏览器环境中运行，通过挂载 SillyTavern 的事件和 API 来扩展其功能。
:::

:::callout
**[服务器插件（Server Plugins）](Server-Plugins)**

服务器插件允许通过在 NodeJS 环境中运行代码来添加功能，例如新的 API 端点。
:::

:::callout
**[国际化（i18n）](i18n)**

了解如何将 SillyTavern 的用户界面翻译为你的语言。
:::
