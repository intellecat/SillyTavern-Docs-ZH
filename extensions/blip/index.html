Blip

本指南将引导你完成 Blip 扩展的设置和自定义，以丰富你的 SillyTavern 体验。该扩展以可变速度为消息文本添加打字动画，并在动画过程中播放音效。你可以使用音频文件，也可以生成声音。

前提条件

在开始之前，请确保满足以下前提条件：

  • 确保你使用的是最新版本的 SillyTavern。
  • 从扩展面板（堆叠方块图标）的"下载扩展与资产"菜单中安装"Blip"扩展。

Blip 全局设置

  1. Blip 用户消息：

    • 勾选复选框以在用户消息上播放动画。
    • 为用户设置一个配置文件，或设置默认配置文件以启用用户的 Blip 动画。

  2. 仅对特定文本启用 Blip：

    • 勾选复选框以仅对引号内的文本播放 Blip 效果。
    • 勾选复选框以忽略星号内的所有内容。

  3. 自动向下滚动：

    • 勾选复选框使聊天跟随文本动画向下滚动；如需在动画期间自由滚动，请取消勾选。

  4. 音频音量

    • 如果只需要文字动画效果，可以将音频静音。
    • 可以调整 Blip 音频的全局音量。

角色动画/语音配置文件

你可以为每个角色保存配置文件：

  • 包括用户及一个可选的默认配置文件（当角色没有配置文件时使用）。
  • 如果列表中只显示当前聊天的角色，点击复选框可显示所有角色。

  1. 选择要分配/更新配置文件的角色：

    • 选择一个角色。如果该角色有配置文件，将自动加载。
    • 如果该角色尚无配置文件，当前参数将成为其配置文件设置。
    • 可以使用删除按钮删除任意配置文件。
    • 如果角色未出现在列表中，请使用刷新按钮。

  2. 文本动画设置：

    • 设置文字速度：每个字母打印之间的毫秒延迟。
    • 设置与 1.0 不同的最小/最大速度倍数以产生随机速度动画效果。
    • 将逗号/句子延迟设置为大于 0 的值，可在打印特殊字符时添加暂停，使动画更生动。此时音频也会暂停。

  3. 音频参数：

    • 设置仅影响该语音配置文件的音量倍数（如需要）。
    • 设置音频速度：每个 Blip 音效之间的延迟，与文字速度无关。

  4. Blip 来源：生成的声音：

    • 使用最小/最大频率滑块自定义播放的 Blip 声音。
    • 若最小/最大值不同，每次播放时会随机选择该范围内的声音。

  5. Blip 来源：文件：

    • 从列表中选择一个文件。
    • 可以从资产扩展菜单获取官方 ST Blip 资产。
    • 或直接将文件放入：\SillyTavern\data\<user-handle>\assets\blip。
    • 如需确保整个文件播放完毕再重新播放，请勾选对应复选框。

感谢你阅读本指南！你的 SillyTavern 体验现已通过文字动画和 Blip 语音得到丰富。
