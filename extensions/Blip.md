# Blip

本指南将引导您完成 SillyTavern 的 blip 扩展的设置和自定义。此扩展以可变速度为消息文本添加动画,并在动画过程中播放声音。您可以使用音频文件或生成声音。

## 前置条件

开始之前,请确保您已满足以下前置条件:

- 确保您使用的是最新版本的 SillyTavern。
- 从扩展面板(堆叠方块图标)中的"下载扩展和资源"菜单安装"Blip"扩展。

## Blip 全局设置

1. **Blip 用户消息**:
   - 启用复选框可在用户消息上播放动画。
   - 如果您希望为用户播放 blip 动画,请为用户设置配置文件或默认配置文件。

2. **仅对特定文本进行 Blip**:
   - 启用复选框可仅对引号内的文本进行 blip。
   - 启用复选框可忽略星号内的所有内容。

3. **自动向下滚动**:
   - 启用复选框可使聊天向下滚动以跟随文本动画,如果您想在动画期间自由滚动,请禁用它。

4. **音频音量**
   - 如果只需要文本动画,请将音频静音。
   - 您可以调整 blip 音频的全局音量。

## 角色动画/语音配置文件

您可以为每个角色保存配置文件:
   - 包括用户和可选的默认配置文件,当角色没有配置文件时将使用该配置文件。
   - 如果列表中仅显示当前聊天角色,请单击复选框以显示所有角色。

1. **选择要分配/更新配置文件的角色**:
   - 选择一个角色,如果他有配置文件,它将被加载。
   - 如果它还没有配置文件,当前参数将成为他的配置文件设置。
   - 可以使用删除按钮删除任何配置文件。
   - 如果您的角色未出现在列表中,请使用刷新按钮。

2. **文本动画设置**:
   - 设置文本速度:每个字母打印之间的延迟(以毫秒为单位)。
   - 将最小/最大速度乘数设置为不同于 1.0 以实现速度动画的随机性。
   - 将逗号/短语延迟设置为大于 0,以在打印特殊字符时添加暂停,可以为动画增添更多活力。在这种情况下,音频也会暂停。

3. **音频参数**:
   - 如果需要,设置一个仅影响此语音配置文件的音量乘数。
   - 设置音频速度:每个 blip 声音之间的延迟,与文本速度无关。

4. **Blip 来源:生成的声音**:
   - 使用最小/最大频率滑块自定义播放的 blip 声音。
   - 如果最小/最大不同,则每次播放此范围内的随机声音。

5. **Blip 来源:文件**:
   - 从列表中选择一个文件。
   - 您可以从资源扩展菜单获取官方 ST blip 资源。
   - 或者直接将文件放入:`\SillyTavern\data\<user-handle>\assets\blip`。
   - 如果需要,启用复选框可强制等待整个文件播放完毕后再次播放。

感谢您遵循本指南!您的 SillyTavern 体验现在通过文本动画和 blip 语音得到了丰富。
