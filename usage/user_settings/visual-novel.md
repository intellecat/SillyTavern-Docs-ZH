# 视觉小说（VN）模式

视觉小说模式是 SillyTavern 中的一种特殊屏幕布局，让你可以与带有立绘（或角色卡图片）的角色聊天，界面风格类似于《心跳文学部》、《灰色的果实》、《Fate: Stay/night》等著名视觉小说游戏。

## 切换视觉小说模式

### 启用视觉小说模式

视觉小说模式是 SillyTavern 的内置功能，可以通过进入*用户设置*（用户设置图标），在「无文本阴影」下方勾选**视觉小说模式**来开启。

![用户设置](/static/vn/vn-mode-toggle.png)

### 禁用视觉小说模式

禁用视觉小说模式的步骤与启用相同，取消勾选「视觉小说模式」即可返回普通聊天界面。

!!!warning 关于 VN 扩展与 VN 模式
某些扩展（如 Prome VN 扩展）在使用其自带的 VN 模式时会自动开启「视觉小说模式」。从*用户设置*菜单启用/禁用 VN 模式也会影响这些扩展。
!!!

## 视觉小说界面

![VN 界面](/static/vn/vn-display.png)

在视觉小说模式下，界面会进行调整以容纳显示在屏幕中央的角色立绘（或角色卡图片）。在多角色群聊中，各角色的立绘会相互分散排布，如下图所示。

![群聊 VN 界面](/static/vn/group-vn-display.png)

### VN 模式与 MovingUI

!!!info
要开启 MovingUI，请进入*用户设置*并勾选 **MovingUI**。注意此功能**仅**在桌面端可用。
!!!

在*用户设置*中启用 **MovingUI** 后，你可以自由移动立绘（或角色卡图片）的位置，将其放置在屏幕上你希望的任意位置。

!!!warning 关于立绘尺寸
如果角色立绘尺寸较大，使用 MovingUI 移动某些立绘可能会比较困难，因为拖拽按钮可能被现有立绘遮挡。当屏幕上有多个角色时，可能需要多次调整才能获得理想的布局。
!!!

![群聊 VN 界面（MovingUI）](/static/vn/vn-group-display-movingui.png)

## 获取角色立绘

可以通过网络搜索现有立绘，例如来自视觉小说或含有视觉小说功能的游戏（如 DDLC 或 CounterSide）的角色立绘。如果你想要的角色尚无现成立绘，还有以下几种途径：

1. 在角色帖子中搜索立绘 ZIP 包或立绘包链接。
    !!!info
    部分机器人创建者在发布机器人时会附带立绘包（可能在同一帖子中，也可能在专属的立绘频道内）。先搜索一下，看看是否已有人为你想要的角色制作了立绘。
    !!!
2. 使用 LoRA 和 Stable Diffusion 自行创作。
    !!!warning
    从头生成立绘非常耗时（尤其是当你的角色没有现成 LoRA，和/或你打算使用的 Stable Diffusion 模型也没有对应 LoRA 时），并且需要足够好的硬件——如果你计划制作 28 个表情而非 6 个，或使用 SDXL 并将立绘提升到更高分辨率，对硬件的要求更高。
    !!!
3. 使用角色卡图片。虽然和立绘效果不同，但至少屏幕上有所显示。不过，VN 模式下不支持多个角色卡图片同时显示。
    !!! 使用 Prome 视觉小说扩展时的角色卡图片
    在 Prome 视觉小说扩展 1.0.6+ 版本中，有一项「将角色卡模拟为立绘」的功能，允许在群聊中将角色卡图片作为立绘使用，从而实现有立绘和无立绘角色共存的群聊。

    ![角色卡群聊](/static/vn/extensions/prome/card-emulation.png)
    !!!

## VN 扩展

### Prome 视觉小说扩展

Prome 视觉小说扩展是由 Bronya Rand 和 Prometheus 开发的受认可第三方扩展，在 SillyTavern 原有视觉小说体验的基础上进一步增强，新增了信箱模式（让视觉小说界面更具「电影感」）、带角色立绘变暗效果的焦点模式、传统 VN 模式（聊天中仅显示最后一条消息）等功能，并持续更新中！

|                              信箱模式                              |                          传统 VN 模式                           |
|:------------------------------------------------------------------------:|:----------------------------------------------------------------------:|
| ![水平信箱模式](/static/vn/extensions/prome/horizontal.png) | ![传统 VN 模式](/static/vn/extensions/prome/single-message.png) |

|                 隐藏对话框                  |                      焦点模式（含立绘变暗）                      |
|:---------------------------------------------------------:|:------------------------------------------------------------------------:|
| ![对话框隐藏](/static/vn/extensions/prome/sheld_hide.png) | ![含立绘变暗的焦点模式](/static/vn/extensions/prome/defocus.png) |

安装 Prome 视觉小说扩展，可以进入`下载扩展和资源`并找到 *Prome Visual Novel Extension*，或按照 [Prome 视觉小说扩展](https://github.com/Bronya-Rand/Prome-VN-Extension?tab=readme-ov-file#installation-and-usage) GitHub 页面上的说明进行安装。Prome 的设置可在*扩展* → **Prome（Visual Novel Extension）**中找到，或通过 🪄（魔杖）菜单访问。
