# 界面自定义

## 界面主题 {#ui-theme}

### 主题管理

主题文件允许您保存、分享和重用界面自定义设置。您可以为不同的心情或用途维护多个主题，并在它们之间即时切换。

* 导入/导出主题文件
* 删除现有主题
* 保存对当前主题的更改
* 另存为新主题

本节中的所有设置都会保存到当前主题。如果您切换主题，这些设置将被新主题的设置所替换。

### 显示设置

这些显示选项影响角色和消息在聊天界面中的呈现方式。

#### 头像样式

在圆形、方形、矩形或圆角方形之间选择。此设置同时适用于用户和 AI 的头像。

#### 聊天样式

| 样式        | 描述                                                                                                                                                                    | [斜杠命令](/For_Contributors/st-script.md#ui-styling) |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------|
| **平铺**   | 简洁连续的"聊天记录"风格，为您的 AI 互动提供一块纯净的画布。                                                                                                              | `/flat`<br>`/default`                                      |
| **气泡**   | "即时通讯"风格，每条消息都有独特的气泡，带有圆角和微妙的 3D 效果。                                                                                                          | `/bubble`<br>`/bubbles`                                    |
| **文档**   | 紧凑、类文档的外观，以文本为核心的布局。隐藏过往消息的头像、时间戳和消息控制按钮。                                                                                            | `/single`<br>`/story`                                      |

### 通知

设置通知弹窗（toast 消息）在屏幕上显示的位置。

* 左上
* 居中上方（默认）
* 右上
* 左下
* 居中下方
* 右下

### 媒体样式

聊天消息中媒体附件（图像、音频、视频）的默认显示样式。将媒体附加到聊天消息的扩展可能会覆盖此设置。也可以通过消息右键菜单中的"切换媒体显示样式"操作，对每条消息手动更改。

* **列表**：以网格样式同时显示所有媒体附件。
* **图库**：以轮播图样式显示媒体附件。

!!!
此设置也会影响内联媒体附件发送到支持的聊天补全来源的方式：列表模式同时发送所有附件，而图库模式仅发送已选中的附件。
!!!

### 主题颜色

自定义每个界面元素的配色方案，打造您的专属主题。可以使用颜色选择器选取颜色，适用的地方还包含透明度选项。

* 主文本
* 斜体文本
* 下划线文本
* 引用文本
* 文本阴影
* 聊天背景
* 界面背景
* 界面边框
* 用户消息
* AI 消息

### 布局与视觉设置

通过这些滑块对界面的视觉呈现进行微调。

* **聊天宽度**：调整聊天窗口宽度（屏幕的 25-100%）
* **字体缩放**：自定义文本大小（0.5-1.5 倍）
* **模糊强度**：控制界面面板模糊度（0-30）
* **阴影宽度**：调整文本阴影强度（0-5）

### 主题开关

这些开关控制各种界面功能和行为。某些选项可以提升低端设备的性能，而其他选项则为聊天界面添加有用的信息或功能。

* **减少动画**：禁用动画和过渡效果
* **无模糊效果**：移除背景模糊以提升性能
* **无文本阴影**：禁用文本阴影效果
* **[视觉小说模式](Visual-Novel.md)**：带背景立绘的紧凑聊天
* **展开消息操作**：始终显示完整的消息右键菜单
* **禅模式滑块**：简化的参数控制
* **疯狂实验室模式**：不受限制的参数范围
* **消息计时器**：显示 AI 响应生成时间
* **聊天时间戳**：显示消息时间戳
* **模型图标**：显示消息所使用的 AI 模型图标
* **消息编号**：显示消息序号
* **隐藏聊天头像**：从聊天中移除头像
* **消息词元数**：显示每条消息的词元数
* **紧凑输入区域**：单行输入（仅限移动端）
* **所有消息显示滑动编号**：在所有消息上显示滑动编号
* **角色快速切换**：为收藏角色提供快速选择按钮
* **头像悬停放大**：悬停头像时的缩放效果
* **标签作为文件夹**：使用标签作为文件夹来组织角色
* **单击编辑**：单击消息即可快速打开消息编辑器

### 自定义 CSS

允许您应用自定义 CSS 样式，进一步自定义聊天界面的外观。

使用 <i class="fa-fw fa-solid fa-maximize" title="展开图标"></i> **展开**以展开编辑器窗口，获得更好的可见性和编辑体验。

如果您切换主题，您的自定义 CSS 将被新主题的自定义 CSS 所替换。如果您希望在切换主题时保留自定义 CSS，请将其保存到某个主题中。

如果您使用了大量自定义 CSS，或希望在多个主题中使用相同的自定义 CSS，非官方的 [CSS 代码片段扩展](https://github.com/LenAnderson/SillyTavern-CssSnippets)可以帮助您管理和组织自定义 CSS。

## 消息提示音 {#message-sound}

要在收到机器人新消息时播放自定义提示音，请替换 SillyTavern 文件夹中以下 MP3 文件：

`public/sounds/message.mp3`

以 80% 音量播放。

如果启用了"[仅后台提示音](index.md#miscellaneous)"选项，则仅在 SillyTavern 窗口**未聚焦**时播放提示音。

## 公式渲染

要启用数学公式渲染，请使用 [LaTeX 扩展](https://github.com/SillyTavern/Extension-LaTeX)。要获取该扩展，您需要通过 SillyTavern 中的"下载扩展与资产"菜单来安装它。

在代码块中使用 `latex` 或 `asciimath` 语言标识符，分别用于 LaTeX 和 AsciiMath 格式。该扩展使用 [KaTeX](https://katex.org/) 进行渲染。

<pre><code>```latex
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
```

```asciimath
int_{-oo}^{oo} e^{-x^2} dx = sqrt{pi}
```</code></pre>

!!!info 弃用说明
旧版 `$` 和 `$$` 包裹语法已不再受支持。请使用以下正则表达式脚本来填充旧语法：

* [$$ - LaTeX](https://github.com/SillyTavern/Extension-LaTeX/raw/refs/heads/main/assets/$$_-_latex.json)
* [$ - AsciiMath](https://github.com/SillyTavern/Extension-LaTeX/raw/refs/heads/main/assets/$_-_asciimath.json)
!!!
