# Slash commands

!!!warning
**这不是一个详尽的列表,因为它很少更新。**

要获取在你的实例中有效的最新命令列表,请在任何 SillyTavern 聊天中使用 `/help slash` 聊天命令。
!!!

- `/help` – 显示帮助消息
- `/api` - 切换到不同的 API
- `/sys` - 以系统叙述者身份发送中性消息
- `/send` - 以用户身份发送消息而不触发生成
- `/sendas` – 以特定角色身份发送消息
- `/comment` - 向聊天添加对 AI 不可见的注释
- `/impersonate` - 让 AI 以你的身份写作,可选 prompt
- `/persona` - 设置你的用户角色
- `/imagine` - 从文本 prompt 生成图像
- `/bg` - 通过文件名设置背景图像
