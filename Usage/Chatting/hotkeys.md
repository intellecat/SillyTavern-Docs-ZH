# 快捷键

**要获取在你的 SillyTavern 实例中有效的最新 HotKeys 列表,请在任何聊天中使用 `/help hotkeys` slash command。**

**移动设备上禁用 Hotkeys。**

## Chat Hotkeys

* Up = 编辑聊天中的最后一条消息
* Ctrl+Up = 编辑聊天中的最后一条用户消息
* Left = 向左滑动
* Right = 向右滑动(注意:当聊天栏中输入了内容时,滑动 hotkeys 被禁用)
* Enter (选中聊天栏时) = 向 AI 发送消息
* Ctrl+Enter = 重新生成最后一条 AI 回复
* Alt+Enter = 继续最后一条 AI 回复
* Escape
  * *(编辑消息时且启用了 Message Edit AutoSave)* = 关闭编辑框。
  * *(AI 消息正在生成或流式传输时)* = 立即停止生成。

## Markdown Hotkeys

需要在"用户设置"选项卡下启用。在聊天栏和标有"M↓"图标的文本区域中有效:

* Ctrl+B = \*\*bold\*\*
* Ctrl+I = \*italic\*
* Ctrl+U = \_\_underline\_\_
* Ctrl+K = \`inline code\`
* Ctrl+Shift+~ = \~\~strikethrough\~\~
