# Mancer

Mancer 是一个大语言模型推理服务，让您可以运行任何您想要的提示词，并且不会审查响应。截至撰写时（2024-11-28），大多数模型需要预先充值余额才能开始聊天，但有一个免费模型。

- [Models](https://mancer.tech/models)
- [Pricing](https://mancer.tech/pricing)

## How to Get Started

1. 在 [mancer.tech](https://mancer.tech/) 注册账户。
2. 点击 Dashboard 并复制您的 API Key。

    ![Mancer dashboard as of 2/27/2024](/static/mancer-dashboard.png)

3. 在 SillyTavern 中，选择 Text Completion API，然后在 API Type 下选择 Mancer。
4. 输入您的 Mancer API Key 并点击 Connect。

![API Key](/static/mancer-key.png)

现在您应该可以与任何您选择的 Mancer 模型聊天了。

## Anonymous Logging

如果您不介意您的聊天可能被用于训练模型、改进 Mancer 的服务、发布数据集或*其他任何*他们可能决定用它做的事情，您可以选择加入匿名日志记录，以获得特定模型 25% 的 token 折扣。只需前往您的 [Mancer dashboard](https://mancer.tech/dashboard.html) 并勾选 `Enable Anon. Logging`。
