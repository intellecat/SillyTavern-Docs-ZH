# 开发与自动化

SillyTavern 的默认包仅提供一个基础平台，可以通过多种方式进行扩展。

:::callout
**[STscript](st-script)**

STscript 是一种基于批处理聊天命令的强大脚本语言，无需任何编程知识即可使用。
:::

:::callout
**[Function Calling](Function-Calling)**

通过让 LLM 使用外部数据源或触发扩展的特定功能来添加更多动态能力。
:::

:::callout
**[UI Extensions](Writing-Extensions)**

UI 扩展在浏览器环境中运行，通过挂钩到事件和 API 来扩展 SillyTavern 的功能。
:::

:::callout
**[Server Plugins](Server-Plugins)**

Server Plugins 允许通过在 NodeJS 环境中运行代码来添加功能，例如新的 API 端点。
:::

:::callout
**[国际化 (i18n)](i18n)**

了解如何将 SillyTavern 的 UI 翻译成您的语言。
:::
